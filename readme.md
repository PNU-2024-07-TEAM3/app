# 📚 Study On

> 스프링부트를 활용한 스터디 모집 & 학습 공유 플랫폼

---

## 🚀 프로젝트 개요

**Study On**은
스터디원을 모집하고, 스터디 룸을 통해 학습 내용을 공유할 수 있는 웹 플랫폼입니다.

단순 모집 게시판이 아니라,
👉 *스터디 생성 → 참여 → 학습 공유 → 지속 관리*까지 이어지는 흐름을 제공합니다.

---

## ✨ 주요 기능

### 📌 스터디 모집

* 스터디 생성 및 모집 글 작성
* 관심 있는 스터디 탐색 및 참여 신청

### 👥 스터디 관리

* 스터디원 관리
* 스터디 상태 및 진행 현황 확인

### 📝 학습 내용 공유

* 스터디 룸 내 게시글 작성
* 공부한 내용 기록 및 공유

### 🔐 사용자 인증

* 회원가입 / 로그인 기능
* 사용자 기반 접근 제어
